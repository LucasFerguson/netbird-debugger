# NetBird Sentinel

Windows-native monitoring and diagnostic tool for NetBird agent failures.

## Setup

1) Create virtual environment

```powershell
python -m venv .venv
```

2) Activate

```powershell
.\.venv\Scripts\Activate.ps1
```

3) Install deps

```powershell
pip install -r requirements.txt
```

## Run (skeleton)

```powershell
python src\main.py
```
